IDS

IDS (Sistema detección de intrusos) es una aplicación usada para detectar accesos no autorizados a un ordenador o a una red, es decir, son sistemas que monitorizan el tráfico entrante y lo cotejan con una base de datos actualizada de firmas de ataque conocidas.

Antivirus

Un antivirus protege tu dispositivo de virus, spyware y otros tipos de malware y virus. Este software realiza exploraciones periódicas para detectar amenazas y vulnerabilidades y proporciona protección contra enlaces infectados con malware en la web.

FireWall/Proxy

Un firewall monitoriza el tráfico entrante y saliente y decide si pasa o no en función de unas restricciones de seguridad ya definidas, estableciendo una barrera entre tu red y cualquier peligro externo.
